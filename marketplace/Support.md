## Report Issues

### Bug Reports

Found a bug? Please report it on GitHub:

**[Report a Bug](https://github.com/alfayedoficial/Astagfirullah-plugin-version/issues/new)**

Please include:
- **Description**: Clear explanation of the issue
- **Steps to Reproduce**: How to trigger the bug
- **Expected Behavior**: What should happen
- **Actual Behavior**: What actually happens
- **Environment**:
  - IDE name and version
  - Plugin version
  - Operating System
- **Screenshots**: If applicable

## Quick Troubleshooting

### Plugin Not Working?

1. **Restart IDE** - Many issues resolve with a restart
2. **Check Compatibility** - Verify IDE version is supported (2023.1 - 2025.3)
3. **Reinstall Plugin** - Uninstall, restart, reinstall
4. **Clear Cache** - Settings → Tools → Astagfirullah → Clear Cache

**[Open a GitHub Issue](https://github.com/alfayedoficial/Astagfirullah-plugin-version/issues/new)**
